# 06 - IDEIAS FUTURAS

**Objetivo:** Guardar ideias para não perder o foco no escopo atual (MVP), mas garantindo que o roadmap não se perca.

## Lista de Ideias
- **Autenticação Avançada:** Implementar NextAuth no Dashboard com JWT (atualmente é apiKey local).
- **Multi-tenant / SaaS:** Expandir as tabelas de banco de dados para suportar agências e múltiplos usuários isolados.
- **Webhook de Retorno:** Notificar sistemas de terceiros (ERP, PDV) quando um Pix for pago.
- **Relatórios Avançados:** Dashboard com gráficos de volume transacionado por banco e taxa de falhas.
- **Módulo de Reconciliação:** Cruzar o extrato bancário com o painel de pagamentos pagos para conferência contábil automatizada.
- **Regras de Auto-aprovação:** Permitir configurar o painel para auto-aprovar Pix de até R$ X para favorecidos pré-cadastrados (White-list).
